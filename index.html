ACCB - Price Miner

Esta aplicação objetivou modernizar o processo de coleta de preços dos produtos que compõem a cesta básica do projeto de extensão intitulado Acompanhamento do Custo da Cesta Básica (ACCB). Para isso foram desenvolvidas duas ferramentas uma delas sendo esta ferramenta. Esta aplicação desktop automatiza a pesquisa na plataforma Preço da Hora – Bahia (precodahora.ba.gov.br) que disponibiliza os preços de produtos em diversos estabelecimentos da Bahia. Esta aplicação desktop atualmente disponível somente para o sistema operacional Windows, com alguns testes realizados na plataforma Linux, formata os dados coletados no processo de scrapping no referido site e devolve uma coleção de arquivos em excel contendo informações de preços de produtos de interesse em diferentes formatação e agrupamento de dados para facilitar o entendimento geral dos dados coletados. Essa ferramenta pode ser utilizada por outras instituições da Bahia, bastando apenas alterar o arquivo de configuração do programa
