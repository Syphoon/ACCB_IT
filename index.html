ACCB - Price Miner

Esta aplicação tem como objetivo modernizar o processo de coleta de preços dos produtos que compõem a cesta básica do projeto de extensão intitulado Acompanhamento do Custo da Cesta Básica (ACCB). O software disponibiliza uma opção digital e semi automática para a coleta de dados de produtos pré-cadastrados no sistema e seus respectivos estabelecimentos. Ao fim de cada pesquisa é disponibilizado uma coleção de arquivos excel com os dados coletados para análises futuras.
