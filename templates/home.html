Dicas para utilização do programa
  • Atualizar a página do programa manualmente irá fechar o programa.
  • Todo o progresso de uma pesquisa é salvo durante a mesma, ou seja, em caso de erro ou perda de internet, é possível retomar uma pesquisa através de um popup que aparece quando o programa é aberto.
  • A quantidade de estabelecimentos cadastrados não influênciam no tempo da pesquisa em si, os arquivos são gerados separadamentes para diminuir o custo de tempo consumido. Em contrapartida cada palavra chave aumenta em media 2-4 minutos no tempo total da pesquisa.
  • É possível pesquisar por qualquer campo da tabela de pesquisas disponível na janela de pesquisas.
  • Evite o máximo atualizar a página manualmente ou fecha-la durante uma pesquisa para minimizar a probabilidade de erro causado durante uma pesquisa.
  • Só é possível abrir uma aba do programa por vez, caso seja aberto uma outra janela você encontrará um aviso o notificando para voltar para a janela principal.
  • Todos os endereços dos estabelecimentos devem ser cadastrados da forma mais identica possível aos valores encontrados no Preço da Hora - Bahia para máximizar as chances de uma filtragem correta.
  • Antes de cadastrar uma cidade, confira no site do preço da hora como o nome do município aparece, isso pode ser feito nos filtros de pesquisa em : Alterar Localização > (popup) Usar centro do município > Insira o nome do município desejado.
